CSS functions.

A CSS function it's a named, self-contained piece of code that complete a specific task.

- url ()

It's use to include a file, the parameter is an absolute URL, a realtive URL, or a data URi. The url () function can be passed as a parameter of another CSS function.

background-image: url(images/canada.jpg);

- attr ()

It's use to retrieve the value of an attribute of the selected element and use it in the stylesheet. It can also be used on pseudo-elements, in which case the value of the attribute on the pseudo-element's originating element is returned.

Let's use it to show the Link inside a anchor

Hi! I have a link inside content: " [ " attr(href) " ] ";

- calc ()

This is a special function for doing basic Math.

For example:
If we want to use for any reason the 100% of available space - X value.

width: calc(100% - 20px);
height: calc(100% - 50px);

- lang ()

Including a lang attribute in your HTML is a really important thing to do. When present in your HTML, you are able to use lang () and apply styles based on it.

Petit a petit, l'oiseau fait son nid

p:lang(fr){
color: rgb(5, 105, 67); }

- blur ()

Applies a Gaussian blur to the input images. It's result is a filter function.

filter: blur(5px);

- opacity ()

Sets the opacity level for the element (transparency level).

filter: opacity(10%);

- brightness ()

Adjust the brightness of the image.

filter: brightness(140%);

- var ()

This CSS function can be used to insert the value of a custom property.
The first argument to the function is the name of the custom property to be substituted. An optional second argument to the function serves as a fallback value.

This text is styled with a var declaration

:root {
--colorExample: rgb(183, 0, 255); }
color: var(--colorExample);

- RGB () & RGBA ()

This function notation expresses a color a ccording to its red, green, and blue components. Also an optional Alpha component represents the color's transparency.

RGB
RGBA transparency
background-color: rgb(234, 123, 232);
background-color: rgba(145 152 84 / 0.3);

- hex

It's based on the same principles of RGB but wrote it in a Hexadecimal notation.

background-color: #324312;
